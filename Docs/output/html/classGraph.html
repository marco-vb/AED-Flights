Detailed Description

Classe usada para representar um grafo dirigido.

Esta classe representa um grafo dirigido (que, neste projeto, representa aeroportos e os voos entre eles) a partir de um vetor de nós (de tamanho n+1, pois o indíce 0 não pertence ao grafo), onde cada nó possui uma lista de adjacências. Os métodos implementados permitem saber várias informações sobre o grafo, como por exemplo o grau de um nó, os menores caminhos entre dois nós (com ou sem distâncias), pontos de articulalção, etc.
